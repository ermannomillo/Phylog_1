# 

Phylog is an embedded system developed for climate monitoring stations. It works on _PhylogOS_: a modified embedded Linux distro.

**Peripherals**
* RJ-45 
* USB
* 83 GPIO
* Dual band Lora modem (868/434 MHz)
* SD card reader
* 8 channel ADC/DAC (12bit resolution)
* RS-232 (UART/SPI/I2C)
* JTAG
